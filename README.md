This project is a full-stack web application built using the MERN stack (MongoDB, Express.js, React, Node.js).
It allows users to browse products, add them to a cart, and place orders.
The backend handles product management, cart logic, and order creation, while the frontend provides a smooth, responsive UI built with Tailwind CSS.

**Tech Stack**
--Frontend--
>React.js
>Tailwind CSS
>Axios
>React Router DOM

--Backend--

>Node.js
>Express.js
>MongoDB + Mongoose
>CORS, dotenv, body-parser

**Features**
✅ Display products with images, prices, and descriptions
✅ Add / remove items from the cart
✅ Adjust item quantity dynamically
✅ Place an order and view order history
✅ Responsive design — works on mobile and desktop
✅ Clean codebase, modular structure
